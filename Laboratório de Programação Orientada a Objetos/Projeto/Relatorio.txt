Relatório

Foi realizado uma abstração para o programa onde os pontos importantes são: Criar ou editar os dados de um time, jogar ou simular até 5 partidas, exibir detalhes do campeonato, exibir uma nota fiscal contendo todos os funcionários do clube, exibir um relatório dos jogadores e sair do programa.
Para que se torne mais fácil de mexer, utilizei a IDE NetBeans para criar as telas personalizadas. Na versão 1.0.0 do programa, foi realizado o design das telas de gerenciamento e cadastro de um time. Na tela de cadastro, o usuário deve inserir o nome do time, a sigla do time e o ano de criação, sendo o limite a data atual (2019). Ao clicar no botão "Criar/Editar", o nome é automaticamente alterado na tela e gerenciamento. Na tela de gerencimento, que será a tela principal do programa, contém os dados de partidas jogadas, pontos e nome do time, além disso conta com as funções de criar ou editar o time, jogar uma partida, simular uma partida, exibir os detalhes do campeonato, emitir uma nota fiscal dos funcionários, emitir um relatório e sair do programa. Na versão 1.0.0 do programa foi realizado a codificação do botão de criação/edição do time, onde ele abre a tela de cadastro e retorna para a de gerenciamento com os dados alterados.
Na versão 2.0.1 do programa foi realizado a criação de design da tela de Jogar Partida, assim como foi criado as funções para todos os botões restantes. Foram criados métodos sets para os pontos e número de partidas jogadas. Também foi feita a validação dos botões de jogar e simular, pois quando atingirem o limite de 5 jogos, não serão mais funcionais.

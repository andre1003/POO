* Java JDK
Fases para desenvolvimento JAVA
1) Criar projeto
  Escrever ou editar um arquivo com um editor. Arquivos de código fonte JAVA recebem a extensão .java

2) Compilar
  Comando JAVAC
Ex.: compilar Teste.java
javac Teste.java
Compilador gera um arquivo.class, convertivo em bytecodes.

3) Carregar
 Executar o .class, carrega na Memória principal -MP.

4) Verificação
  A Máquina Virtual JAVA - JVM carrega e executa o bytecode verificando violações de segurança.

Teste:
public class Teste{

		System.out.println("Ola Mundo!");
	}
